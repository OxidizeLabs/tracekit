# Rust Project Template Docs

This is a minimal documentation site for the Rust project template repository.

## Getting Started

- Use GitHub “Use this template” to create a new repository.
- Update `Cargo.toml` metadata and rename the crate.
- Replace `README.md` with your project’s description and usage.

## Local Development

```sh
cargo fmt
cargo clippy --all-targets --all-features
cargo test --all-targets --all-features
cargo run
```
